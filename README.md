**CodeAlpha_BasicNetworkSniffer**

A Python-based network packet sniffer for capturing, analyzing, and monitoring network traffic. Built using Scapy to explore packet structures, protocols, and basic network analysis concepts.

**Project Overview**

This project focuses on understanding how data travels across networks by capturing packets and analyzing their structure, protocols, and communication patterns.

**Objectives**

- Learn packet capture techniques
- Understand network protocols
- Analyze packet metadata
- Explore basic network monitoring concepts

**Technologies Used**

- Python
- Scapy
- Wireshark
